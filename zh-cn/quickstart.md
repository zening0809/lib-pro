如何查看使用的 nv-lib 版本？



## 如何安装

**npm 安装**
```shell
$ npm install nv-lib | yarn add nv-lib
```

**全局安装**
```shell
npm install nv-lib -g | yarn add nv-lib -g
```

## 如何引用

> 浏览器环境

```Javascript
import _nv from 'nv-lib' | import _nv as _ from 'nv-lib'
```

> node 环境

```Javascript
const _nv = require('nv-lib')
```

> typescript 中引入

```Javascript
import _nv from 'nv-lib' | import _nv as _ from 'nv-lib'
```
