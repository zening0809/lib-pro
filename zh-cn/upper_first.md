`upperFirst` 首字母大写

> 首字母大写（upperFirst）

> *描述*

```javascript
转换字符串string的首字母为大写。
```

> *入参*

```javascript
[string=''] (string): 要转换的字符串。
```

> *返回值*

```javascript
(string): 返回转换后的字符串。
```

> *类型定义*

```javascript
type upperCaseType = (v: string) =>  string;
```

> *例子*

```javascript
_nv.upperFirst('fred');
// => 'Fred'
 
_nv.upperFirst('FRED');
// => 'FRED'
```
